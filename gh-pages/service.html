Creating ServicesÂ¶

After you setup your first Project (See Setup a Pyramid application) you can define a service. Create a view.py file and add it to the package you created:

from lovely.pyrest.service import Service

svc = Service(name='svc', path='/svc')


@svc.get()
def get(request):
    return {'title': 'Hello World!'}

With this application you can perform GET requests on the defined endpoint:

>>> print app.get('/svc')
Response: 200 OK
Content-Type: application/json; charset=UTF-8
{"title": "Hello World!"}

The default renderer of a Service is json. You can overwrite it in the method decorator:

@svc.get(render='string')
def get(request):
    return 'Hello World!'

Generally you can set every predicate argument which you can use in pyramid. See Predicate Arguments.
